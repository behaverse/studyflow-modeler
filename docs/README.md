# Studyflow Modeler Docs

This documentation site is built with [Quarto](https://quarto.org/).

## What is documented here

- **Guides** explain how to create and edit studyflows in the Modeler.
- **Reference** documents the language, element types, schemas, and views.
- **Examples** show real studyflow patterns for analysis, reporting, publication, and related workflows.

## Prerequisites

Install Quarto CLI: <https://quarto.org/docs/get-started/>

```bash
brew install quarto   # macOS
```

## Development

```bash
npm run docs          # preview with live reload
```

## Build

```bash
npm run docs:build    # renders to dist/docs/
```
