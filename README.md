
# Behaverse Studyflow Modeler

Studyflow Modeler is a tool to design and run cognitive experiments using [Studyflow diagrams](https://behaverse.org/projects/studyflows).

## Development

If you already have Node.js installed, you can start the modeler app by running:

```bash
npm install
npm run dev
```

To run the documentations locally, use:

```bash
npm run docs
```
